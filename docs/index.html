UNIOPS

Agent-First Bastion Host Platform

Features

  • SSH Proxy with Auto Key Deployment
  • Agent Management (Claude, Codex, Custom)
  • Session Recording & Audit
  • Web Terminal with xterm.js
  • ROG Dark Theme UI

Quick Start

# Docker
docker-compose up -d

# Or build from source
CGO_ENABLED=0 go build -o uniops ./cmd/uniops
./uniops
